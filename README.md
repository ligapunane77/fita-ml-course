# Rīgas dzīvokļu tirgus analīze klienta vajadzībām ar Machine Learning

## Projekta mērķis
Analizēt Rīgas dzīvokļu tirgu un palīdzēt klientam atrast
piemērotāko dzīvokli ar Machine Learning metodēm.

## Klienta profils
- 2 istabu dzīvoklis, 45–50 m²
- Budžets: €65 000–75 000
- Vēlamie rajoni: Imanta, Dārzciems, Teika, Jugla, Mežciems

## Izmantotās metodes
- Regression — dzīvokļa cenas prognozēšana
- Classification — premium dzīvokļu noteikšana
- Clustering — tirgus segmentācija

## Failu struktūra
- `final_project.ipynb` — galvenais notebook
- `riga.csv` — sākotnējie dati
- `riga_clean.csv` — attīrītie dati
- `requirements.txt` — nepieciešamās bibliotēkas

## Bibliotēkas
pandas, numpy, matplotlib, seaborn, scikit-learn
